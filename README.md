**🎮 ボム兵を分けろ！**
爆走するボム兵たちを、赤ゾーンと黒ゾーンに正しく仕分けるシンプルな反射神経ゲーム！

## 📘 ゲーム概要
「ボム兵を分けろ！」は、画面内をランダムに動き回る 赤ボム と 黒ボム を、
正しいゾーンに ドラッグして放り込む スピードアクションゲームです。
赤ボム → 左の赤ゾーンへ
黒ボム → 右の黒ゾーンへ
間違って反対側のゾーンに入れたり、
爆発（タイム切れ）させたりすると ゲームオーバー！
制限時間内に、できるだけたくさんのボムを正しく仕分けして
高スコア・高コンボ を目指します。

## 🕹️ 操作方法
操作	内容
クリック / タップ	ボムを掴む
ドラッグ	掴んだボムを移動させる
離す（ドロップ）	ボムを投げる（慣性あり）
STARTボタン	ゲーム開始
音 ON/OFF	効果音のミュート切り替え

## 🎯 ルール
✔ 正しく仕分けると
スコアUP
タイムが少し回復
コンボ増加（連続成功でボーナスUP）
❌ 間違うと
即ゲームオーバー
（反対ゾーンに落とす / 爆発する）

## 💣 爆弾（ボム兵）の動き
ランダム挙動（風・バウンド・加速）
一定時間で fuse（導火線）が尽きて爆発
掴むと fuse の減りが少しだけ遅くなる
特殊個体 fast は fuse が早く減る

## ⏱️ タイム制限
初期時間：45秒

正しい仕分けでタイムが回復するため、
連続で捌ければ長くプレイできます。

## 💥 コンボシステム
連続で正しく仕分けるほどコンボが上昇
コンボが高いほど スコア倍率 が増加
時間経過で少しコンボが下がる（設定可）
