## ボム兵を分けろ！🧨
HTML5 Canvas と JavaScript で作られたブラウザゲーム
マリオシリーズの「ボム兵」をモチーフにした仕分けゲーム

### ゲーム内容
画面上から出現するボム兵を正しい色のゾーンに仕分ける
赤いボム兵 → 左の赤ゾーン
黒いボム兵 → 右の黒ゾーン

### 操作方法
マウスまたはタッチでボム兵を掴む
ドラッグして下のゾーンに移動
離すと投げることができる

### ルール
制限時間は 45秒
ボム兵には導火線があり、時間切れで爆発するとゲームオーバー
間違った色のゾーンに入れると即ゲームオーバー
正しく仕分けるとスコアが加算され、連続成功でコンボが増加

### スコア
基本スコア：1体につき 10点
コンボ数に応じてボーナス加点あり

### 技術要素
HTML / CSS / JavaScript のみで動作
Canvas 2D API を使用
requestAnimationFrame によるゲームループ
マウス・タッチ両対応
